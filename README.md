# FinançasCasa

Um painel financeiro inteligente com IA para gerenciar suas finanças domésticas brasileiras.

## Começando

### Como abrir?

1. Abra o arquivo `index.html` em seu navegador (Chrome, Firefox, Safari, Edge)
2. Não requer instalação ou internet (funciona offline, exceto IA)
3. Seus dados são salvos automaticamente no navegador

### Funcionalidades principais

**💰 Dashboard**
- Visão geral do saldo, receitas e despesas
- Saúde financeira em tempo real
- Gráficos de fluxo de caixa

**📤 Importar**
- Upload de extratos bancários (PDF, CSV, TXT)
- Classificação automática com IA
- Suporta: Nubank, Inter, Itaú, Bradesco, etc

**📊 Despesas**
- Histórico completo de transações
- Filtrar por categoria
- Editar ou deletar transações

**💵 Orçamento**
- Definir limites por categoria
- Acompanhamento visual em tempo real
- Alertas quando ultrapassa limite

**📈 Investimentos**
- Acompanhe sua carteira
- Sugestões de alocação
- Patrimônio total

**🎯 Metas**
- Crie metas financeiras
- Acompanhe progresso
- Sugestões de metas

**🤖 Consultor IA**
- Chat com Claude (Anthropic)
- Análise personalizada
- Recomendações inteligentes

**⚙ Configurações**
- Chave API da IA
- Backup e restauração
- Reset de dados

## Usando com IA

Para usar o Consultor IA e classificação automática:

1. Vá em **Configurações**
2. Cole sua chave API do Claude (obtém em https://console.anthropic.com)
3. Clique em "Salvar Chave"
4. Volte e use a IA!

## Dados e Privacidade

- ✅ Todos os dados são salvos **localmente** no seu navegador
- ✅ Nada é enviado para servidores (exceto requisições à API da IA)
- ✅ Faça backup regularmente em Configurações
- ✅ Você controla e pode deletar tudo quando quiser

## Suporte

Problemas? Verifique:
- JavaScript está ativado no navegador?
- LocalStorage está habilitado?
- Chave API é válida? (começa com `sk-`)
- Tente atualizar a página (F5)
